This project builds premium local service-business website previews.

Never generate plain template-looking pages.

Design standard:
- Looks like a $1,500–$2,500 local service website
- Hero must feel photographic, bold, and industry-specific
- Dark overlay hero
- Large confident headline
- Real CTA hierarchy
- Trust chips visible above the fold
- No obvious gray placeholders
- If images are not available, use styled image blocks that look intentional
- Strong spacing, strong typography, strong contrast
- Mobile-first but desktop must look polished

Current goal:
V3 should visually match the premium tree-service mockup standard:
- large hero
- professional navigation
- service cards
- split trust section
- stats/testimonial block
- final CTA
- footer

Avoid:
- generic SaaS layouts
- tiny text
- empty whitespace
- placeholder-looking boxes
- unrelated stock imagery
